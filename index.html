CORS Fix — Required for Browser Fetch
The browser will block requests to Google Apps Script unless your doGet() function returns JSONP. This portal uses a no-library JSONP loader — no npm or CDN needed. Add this wrapper to your Apps Script:
function doGet(e) {
  const cb = e.parameter.callback;
  const data = JSON.stringify(getSheetData()); // your function
  return ContentService
    .createTextOutput(cb + '(' + data + ')')
    .setMimeType(ContentService.MimeType.JAVASCRIPT);
}
📋 Apps Script Setup — 3 Steps
Step 1 · Script
In your Google Sheet go to Extensions → Apps Script. Paste the Code.gs for your WSAI module.
Step 2 · Deploy
Click Deploy → New Deployment. Set type to Web App, execute as Me, access to Anyone within IIT Madras. Copy the Web App URL.
Step 3 · Paste & Fetch
Paste the URL in the module selector above. Click ⬇ Fetch All Sheets to pull live data into the portal.
